 课书(CourseBook)特色功能是自动选课，为保证公平性，课书目前是少量开放的。只用于帮助部分因为时间、网络或设备等原因不能上线抢有余位的课程，但又急需或者心仪某门课程的同学。

 课书使用教务系统帐号和密码登录。beta版具有两个功能：空位选课和空位先退后选。前者适用于没有选满课，还有选课空余的同学，后者适用于已经选满了课，但对其中一门不够满意，想要更换一门已经满人的课的同学


 步骤教程：

 1. 登录

 2. 选择课程分类

 3. 点击课程名，选择你要抢的心仪课程
 4. 【可选】当上述课程有空位时，你希望退掉哪门课，然后去选第3步的课程。(失败立刻选回原课程)

 5. 关注微信公众号。发送你的学号，进入后台查询状态。成功则结束。
    【注意】如果查询状态是"选课中"，请勿自行登录教务系统，否则后台无法更新状态信息。
